# 汉字认字乐园 (Chinese Flashcards)

这是一个专为海外华人小朋友设计的汉字认字卡网站。基于“马立平中文”教材（1-7年级），帮助孩子们在课后高效复习和巩固生字。

## ✨ 核心功能
*   **按年级与课文分类**：内置 1-7 年级所有课文的生字与词组，随时选择需要复习的内容。
*   **三种复习模式**：支持“当前课”、“年级复习”、“总复习”三种模式，灵活满足学习需求。
*   **智能生字本**：自动记录“不认识”和“模糊”的生字到右侧生字本，通过缓存记忆学习状态，点击“认识”后即可移出。
*   **标准语音播报**：点击“读音与组词”，自动朗读当前生字以及内置的 3 个扩展组词，加深词汇印象。
*   **互动书写板**：集成 Hanzi Writer 引擎，支持“播放笔顺”动画演示，并提供“自己写”的手写测验模式（写对绿光，写错红光）。
*   **纯前端架构**：无需任何后端服务器或数据库，纯 HTML/CSS/JavaScript 构建，极速响应，支持离线或静态页面托管。

## 📂 项目结构
```text
chinese-flashcards/
├── index.html       # 选择年级的入口主页

├── css/
│   └── style.css    # 响应式布局与现代化UI样式
└── js/

        ├── grade1.js
        ├── grade2.js
        └── ...
```

## 🚀 本地运行与部署
本网站为纯静态资源，**无需安装 Node.js、Python 等环境**。

### 本地直接运行
直接双击打开 `index.html` 即可在浏览器中体验。
*(注：如果由于部分浏览器的严格安全策略导致跨域报错，请使用 VSCode 的 Live Server 插件或在目录下执行 `python -m http.server` 来启动本地服务器运行)*

### GitHub Pages 部署 (推荐)
本项目极其适合托管于 GitHub Pages 进行全网免费访问，详情请参考项目部署指南。

## 💡 技术栈
*   HTML5 + CSS3 (Flexbox/Grid, 响应式设计)
*   Vanilla JavaScript (ES6)
*   **pinyin-pro**: 动态拼音解析
*   **Hanzi Writer**: 汉字笔顺动画与手写判定
*   **Web Speech API**: 浏览器原生语音合成引擎
